お問い合わせ

よくある質問

お客様から寄せられる質問と回答をまとめました

製品について

Lixel L2 ProとLixel K1の違いは何ですか？

L2 Proは長距離・高精度スキャンに特化したプロフェッショナルモデルで、最大300mの測定範囲と32チャンネルLiDARを搭載しています。一方、K1は1kg未満の超軽量コンパクトモデルで、40mの測定範囲で手軽に高品質なスキャンが可能です。用途に応じてお選びください。

スキャナーの精度はどの程度ですか？

L2 Proシリーズは相対精度±2cm（100m以内の任意の2点間）、絶対精度±3cm（RTK使用時）を実現しています。K1は相対精度±1.2cm、絶対精度≤3cmです。いずれも測量グレードの高精度を提供します。

バッテリーの持続時間はどのくらいですか？

L2 Proは標準バッテリーで約5時間、K1は約3時間の連続稼働が可能です。追加バッテリーもご用意しており、ホットスワップ対応のため作業を中断することなく交換できます。

防水・防塵性能はありますか？

両製品ともIP54相当の防塵・防滴性能を持っています。軽い雨や埃の多い環境でも使用可能ですが、完全防水ではないため水没にはご注意ください。

購入・価格について

価格を教えてください

製品価格は構成やオプションにより異なります。お客様の用途に最適な構成をご提案させていただきますので、お問い合わせフォームよりご連絡ください。見積もりは無料です。

レンタルやリースは可能ですか？

短期レンタルおよび長期リースプランをご用意しています。プロジェクト単位でのご利用や、導入前のお試し利用にもご活用いただけます。詳細はお問い合わせください。

納期はどのくらいですか？

在庫状況により異なりますが、標準構成の場合は通常2-3週間程度でお届けしています。カスタム構成や大量注文の場合は別途ご相談ください。

支払い方法は何が利用できますか？

銀行振込でのお支払いに対応しています。分割払いやリース契約のご相談も承っております。

使い方・操作について

初心者でも使えますか？

はい、直感的な操作設計により初心者の方でも簡単にご利用いただけます。Lixel GOアプリでワンタッチ操作が可能で、日本語のマニュアルとチュートリアル動画もご用意しています。

スキャンデータの処理にはどんなソフトが必要ですか？

Lixelスキャナーには無料のLixel Studioが付属しており、点群データの表示、編集、エクスポートが可能です。より高度な3D再構築にはLixel Cyber Color Studio（別売）をおすすめします。

スマートフォンやタブレットで操作できますか？

はい、iOS/Android対応のLixel GOアプリを使用して、スマートフォンやタブレットから操作できます。リアルタイムプレビューや設定変更も可能です。

他社製ソフトウェアとの互換性はありますか？

標準的な点群フォーマット（LAS、LAZ、PLY、E57、PCD等）でエクスポート可能なため、AutoCAD、Revit、CloudCompareなど主要なCAD/BIMソフトウェアで利用できます。

ファームウェア更新関連

K1のファームウェアをUSB経由で更新するには？

1. バッテリーを70%以上充電 2. 公式サイトからファームウェアをダウンロード（解凍しない） 3. LixelGoでUSBモードを有効にし、PCと接続 4. ファイルをUSBドライブのルートにコピー 5. 安全に取り外し後、再起動 → 赤緑交互点滅でアップデート開始 6. 緑点灯で完了。LixelGoでバージョン確認

L2 Proのファームウェア更新手順も同様？

はい。手順はK1と共通で、L2 ProでもUSBモード→再起動で更新されます。

カメラ・解像度関連

K1のカメラ解像度は？

• モノクロカメラ：1MP（2基） • カラーカメラ：48MP（2基） • パノラマ合成画像の解像度：最大56MP

L2 Proのカメラ構成は？

• 前方：モノクロ 1MP • 両側：カラー 48MP ×2

デバイス環境仕様・保守

使用可能な温度範囲は？

-20°C〜50°C の範囲で安定動作します。それ以外は故障・精度劣化の可能性があります。

防爆（爆発防止）性能はある？

対応していません。

LiDARスキャナの清掃方法は？

付属の不織布で、同一方向にやさしく拭いてください。

LixelGoアプリ仕様・対応端末

アプリ対応OS・端末要件は？

• Android：Ver10以上、Snapdragon 8推奨、RAM 8GB以上 • iOS：iOS 17.5.1以上、iPhone13以降推奨（Type-C推奨）

ホットスポットが頻繁に切れる？

スマホの高温による保護機能で自動切断されることがあります。他機器の干渉、VPN、通信制限にも注意してください。

RTK設定・運用

RTKの「1+1モード」には対応している？

現時点では非対応です。

RTK設定の変更方法は？（L2 Pro）

1. LixelGoの「高度な設定」を有効化 2. RTK種別を「Max RTK」に設定（サーベイ用アンテナ使用時） 3. 標準アンテナへ戻す場合は「Close」に切り替え

RTKアカウントの取得方法は？

GNSSサービス事業者へ申請し、LixelGo内のRTK設定でログインしてください。

その他便利なTips

LCCモデルのウォーターマークはどうすれば消える？

LCCやPLY形式でエクスポートすると自動的に消えます。

「No LCC Reconstruction Data」エラーの対処法は？

結果データを移動した場合、元の位置に戻すか、「Import Capture」で再読み込みしてください。

LCCでクロップ（切り出し）したモデルを他形式に変換可能？

LCC⇔PLYの相互変換が可能です。切り出し後は新規保存が必要です。

電源・スキャン制限に関する注意事項

スキャン中に電池が切れるとどうなる？

バッテリー残量10%で警告が出ます。そのまま継続するとデータが破損し、処理不能になるリスクがあります。

バッテリー寿命の目安は？

約300〜500回の充放電で劣化し、容量が70〜80%になると寿命です。使用後は速やかに充電し、長期間使わないときも定期的な充電が必要です。

LCC収録時間に上限はある？

最短3分、最長30分。30分を超えると動画が分割されてしまい、復元不可能です。収録は一度で完結させることが重要で、動画のマージ機能はありません。

制御点・測量関連

壁面に制御点を取ることは可能？

物理的には可能ですが、姿勢警告が出たり精度が落ちる可能性があるため非推奨です。どうしても必要な場合は、丁寧に傾けて振動を避けてください。

同じ制御点を複数回取ってもいい？

問題ありません。同名で取るとソフト側が自動で認識・統合してくれます。実座標CSVには1回だけ記載すればOKです。

LCC関連（モデル処理）

LCC Studioでログインしないと使えない？

スタンダード版はオンラインログイン必須です。永久ライセンス版はログイン不要で使用可能です（営業担当にお問い合わせください）。

LCCモデルが「再構築待機中」のまま動かない？

保存先ディレクトリに空き容量がない可能性があります。他のパスに変更して再構築を試してください。

LCCでエラーコード 0xc0000409 / 0xc0000005 が出る？

Intel第13/14世代CPUによるオーバークロックの影響があります。CPUのダウンクロック設定を行うか、別PCで試してください。

LCCモデルがぼやける原因は？

カメラ設置方向の誤り、映像と点群が別のものになっている、反射や露出過多、撮影角度不足などが原因です。

LCC Viewerで表示できるブラウザは？

Chrome、Firefox、Edge、Safari に対応しています。

LCCモデルをOBJやOSGBに変換できる？

LCCモデルはメッシュ形式ではないため、変換は非対応です。

その他技術仕様・用語

Feature Point（特徴点）とは？

壁、柵、屋根などの形状的な目印で、アルゴリズムが認識しやすいものです。特徴点が少ない場所では、動きを工夫したり人工物を追加する工夫が必要です。

SLAMとは？

同時位置推定と地図作成。点群の重なり部分から姿勢を推定・合成して一つのモデルにまとめる手法です。

Poses.csvとGNSS.csvの内容は？

• Poses.csv：タイムスタンプ、XYZ座標、回転の四元数（qw, qx, qy, qz） • GNSS.csv：0＝無効、1＝単独、2＝疑似距離差、4＝固定解、5＝浮動解

アクティベーション関連

ライセンスが失効・認証失敗になるのはなぜ？

管理者権限でのインストール＆起動が必要です。Windowsの日時設定を「今すぐ同期」で修正後、再起動してください。

「正しいSNを入力してください」と表示される理由は？

借用品や未購入機器には試用コードが必要です。SNコードの誤入力にも注意（Oと0、スペース混入など）してください。

バージョン2.5.2から3.0へ切替時の注意点は？

新規アクティベーションコードが必要（1台1コード）です。旧バージョンのプロジェクトファイルは開けないため、元データからLASとして再読み込み推奨です。

動作・クラッシュ関連

Windows 11でソフトが開けない/クラッシュする？

UTF-8を有効にし、「WMIC」のインストール（オプション機能から）を行ってください。

0xc0000142 エラーの対処法は？

UTF-8の未設定が原因です。設定後に再起動してください。

「msvcp140.dll が見つからない」と表示される場合は？

Microsoft公式のランタイム再配布パッケージをインストールしてください。詳細はダウンロードページをご確認ください。

プロジェクト処理中のエラー

LIO/SLAM/最適化エラーの対処法は？

「Robustモード」で再実行してください（精度はやや低下）。初期化時の動き・視野障害に注意し、プロジェクトはCドライブ以外へ配置し、UTF-8を有効にしてください。

メモリ不足/クラッシュが発生する？

RAM 64GB以上推奨です。プロジェクトサイズの5〜465倍の空き容量が必要（L2 Proの場合）です。

地図融合・メッシュ関連

地図の結合（Map Fusion）の方法は？

RTK固定・GCP（制御点）・共通制御点・再開スキャンのいずれかを使用し、15～30mの重複軌道を含めることを推奨します。

メッシュ生成できない・失敗する？

「パノラマ画像の出力」を有効にし、同じLSバージョンでの処理結果を選んでください。L2 Proではアップサンプリング済みのデータは非対応です。

写真・パノラマ・カラー処理

撮影画像の出力方法は？

Lixel Studioでカラー処理後、「panoramicImage」フォルダに保存されます。

パノラマオーバーレイが使えない？

フォルダ名に注意し、処理結果と一致しているか確認してください。

「画像と姿勢が見つからない」と表示される？

パノラマ画像の生成忘れ、もしくはディレクトリ選択ミスが原因です。

ボリューム・測定・RTK関連

体積や重量の計算は可能？

密度を入力することで自動計算が可能です。

高さ異常（ジオイド高）とは？

楕円体高と正高の差で、変換時に必要な情報です。

RTK設定が無効になる理由は？

設定ミス、または未登録のRTKアカウント使用が原因です。

その他操作・注意点

AutoCADや3ds Maxに取り込むには？

LAS形式は直接非対応です。RCP形式に変換が必要です。

USBモードが起動できない・青点灯しない？

再起動＆再接続後、再度USBモードを有効化してください。

SDカードが認識されない・マウントエラー？

SDカードの再挿入＆再起動を試してください。

モバイルアプリ（Lixel Go）

接続エラーが出る・ホットスポットが切れる？

他機器の干渉、スマホの熱暴走、VPN設定に注意してください。iPhoneは「互換性を最大化」をオフにしてください。

撮影中に点群や映像が表示されない？

表示されなくてもデータは保存済みなので継続OKです。

ハードウェア仕様・更新

バッテリー寿命・充電管理の注意点は？

約90分稼働。300～500回の充放電で劣化します。使用後はすぐ充電し、長期保管時は定期的な再充電を推奨します。

ファームウェアのアップデート方法（USB）

ダウンロード → USBモード → 本体にコピー → 再起動でアップデートされます。

トラブルシューティング

スキャンデータにノイズが多い場合の対処法は？

環境光の影響、反射面、振動などが原因の可能性があります。スキャン速度の調整、IMUキャリブレーション、適切なスキャンパスの選択により改善できます。詳しくはサポートまでご相談ください。

RTKが接続できない場合は？

RTK基地局との通信設定、NTRIP設定、SIMカードの状態を確認してください。また、空が開けた場所での使用を推奨します。設定ガイドはダウンロードページからご覧いただけます。

ファームウェアのアップデート方法は？

Lixel GOアプリまたはPCソフトウェアから簡単にアップデートできます。アップデート前には必ずデータのバックアップを取ることをお勧めします。

保証期間と修理について教えてください

製品保証期間は購入日から1年間です。延長保証プランもご用意しています。修理は国内サービスセンターで対応し、代替機の貸出サービスも行っています。

その他

トレーニングやサポートは受けられますか？

導入時の初期トレーニング、定期的なウェビナー、オンサイトトレーニングをご提供しています。技術サポートは電話、メール、リモートサポートで対応いたします。

デモンストレーションは可能ですか？

はい、実機でのデモンストレーションを承っています。お客様の現場での実証実験も可能です。オンラインデモもご用意していますので、お気軽にお申し込みください。

海外での使用は可能ですか？

はい、世界各国でご利用いただけます。ただし、一部の国では電波法規制により使用が制限される場合があります。詳細はお問い合わせください。
